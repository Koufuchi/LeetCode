※ 動態規劃的的四個解題步驟是：

    1. 定義子問題

           將一個問題切分成同樣的好幾個小問題。

           例如  1, 2, 3, ......, n-1, n

           =>   1, 2, 3, ..., k    拆成比 n 小但做法一樣的好幾個小問題

           原問題要能由子問題表示。例如這道小偷問題中，「k = n」 時實際上就是原問題。 
           否則，解了半天子問題還是解不出原問題，那子問題等於白解了。

           一個子問題的解要能通過其他子問題的解求出。例如這道小偷問題中，
           
           f(k) 可以由 f(k-1) 和 f(k-2) 求出，具體原理後面會解釋。
           
           這個性質就是教科書中所說的「最優子結構」。如果定義不出這樣的子問題，那麽這道題實際上沒法用動態規劃解。

    2. 寫出子問題的遞推關系

          在寫遞推關系的時候，要注意寫上 k = 0 和 k = 1 的基本情況：

            當 k = 0 時，沒有房子，所以 f(0) = 0
            當 k = 1 時，只有一個房子，偷這個房子即可，所以 f(1) = nums[0]

    3. 確定 DP 數組的計算順序

    4. 空間優化（可選）
